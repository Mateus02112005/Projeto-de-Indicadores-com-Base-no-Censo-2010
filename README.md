# Projeto de Indicadores com Base no Censo 2010
Este repositório reúne indicadores sociais desenvolvidos na disciplina Prática Estatística 1.
O projeto contém indicadores que foram construídos a partir de dados coletados pelo Censo Demográfico de 2010, disponibilizado pelo IBGE.
Os indicadores aqui apresentados abrangem temas como condições socioeconômicas, infraestrutura domiciliar, desigualdades étnico-raciais e dinâmicas demográficas. Esses dados oferecem uma visão ampla da realidade municipal e auxiliam no planejamento e na formulação de políticas públicas voltadas à redução das desigualdades e à melhoria das condições de vida da população.
